# LankaTV

A Tivimate-style IPTV player for Android TV / Google TV with Electronic Program Guide (EPG), catchup TV, and D-pad remote navigation. Built with Flutter.

[![Download](https://img.shields.io/badge/Download-Latest%20Release-blue)](https://github.com/AtlasLK/LankaTV/releases/latest)
[![Website](https://img.shields.io/badge/Website-lankatv.vercel.app-green)](https://lankatv.vercel.app)

## Features

- 📺 **Live TV Streaming** — HLS/M3U8/DASH playback with native ExoPlayer
- 🔐 **DRM Support** — Widevine DRM for protected content
- 📋 **Electronic Program Guide (EPG)** — XMLTV-based guide with day tabs, program images, and time indicators
- ⏪ **Catchup TV** — Watch programs from the last 72 hours + next day
- 📂 **Channel Categories** — Sidebar with All Channels, Favorites, Recent, Categories, Catchup, and EPG Guide
- ⚙️ **Player Settings** — Buffer size, aspect ratio, and decoder options
- 🎨 **Appearance Settings** — Toggle channel logos and numbers
- 🎮 **TV Rempte Navigation** — Full android TV remote control support
- 📱 **Google TV Integration** — Leanback launcher support with TV banner

## Usage

1. Launch the app and enter your M3U playlist URL
2. EPG URL will be auto-detected if available in the playlist
3. Browse channels by category and start watching

### Remote Controls (Android TV)

| Key | Action |
|-----|--------|
| Left/Right | Toggle sidebar + channel list |
| Up | Open catchup programs |
| OK/Select | Play selected channel |
| Back | Close overlay |
